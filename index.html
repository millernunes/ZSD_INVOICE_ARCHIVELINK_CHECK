Welcome SAP SD Invoice ArchiveLink Check Project

With the advent of Electronic Invoices the government's started to request that the company's store the Print Invoice Documents for a determined period of time. So, some companies begun to use SAP ArchiveLink Standard solution in order to store the invoices.

But, with the passing of years, some companies realize that some documents were archived more than once or, even worse, they weren't archived at all. This situation occurs because the companies didn't create mechanisms that allows them archive all Invoice documents and that avoid those documents are archived more than once.

The Idea of this project is create a new tool that enables the companies identify which documents are archived, and how many times they are stored.

How to Install ArchiveLink Check in your ECC

In order to install ArchiveLink Check program, you should have already installed SAPLink plugin in your ECC. SAPlink is a project that aims to make it easier to share ABAP developments between programmers. It provides the ability to easily distribute and package custom objects. Please visit the SAPLink Project page for further information.

You can also consult the SAPLink User Documention to now more about this tool.
